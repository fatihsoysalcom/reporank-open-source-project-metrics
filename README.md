# RepoRank Open Source Project Metrics

This Python script demonstrates a simplified RepoRank concept by fetching GitHub repository metrics like stars, forks, and recent commit activity to calculate a basic impact score. It uses the GitHub API to gather data for open-source projects.

## Language

`python`

## How to Run

1. Ensure you have Python installed.
2. Install the 'requests' library: pip install requests
3. Run the script: python repo_ranker.py
